Primeira vez no computador:
1. Instale o Node.js, se ainda nao tiver:
   https://nodejs.org

2. Instale a ferramenta do Firebase:
   npm install -g firebase-tools

3. Entre na sua conta Firebase:
   firebase login

4. Entre nesta pasta:
   cd "C:\Users\Pichau\Documents\Codex\2026-05-12\files-mentioned-by-the-user-supervisiorio\qta-mobile-publicar"

5. Publique:
   firebase deploy --only hosting

Depois da primeira publicacao:
- Se mudar apenas dados no Firebase, nao precisa publicar de novo.
- Se mudar o codigo da pagina, rode novamente:
  firebase deploy --only hosting
